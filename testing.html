STEP 2Install & Launch

Download ArtDrop for your platform and run it. The app opens in your browser at localhost:5100.

  • Download and run ArtDrop Mac: double-click the .app file. Windows: run the .exe. If your OS warns about an unrecognized developer, click "Open Anyway" (Mac: System Settings → Privacy) or "Run Anyway" (Windows). ✓ Browser opens to localhost:5100 with ArtDrop dashboard
  • Verify the dashboard loads You should see the ArtDrop UI with a setup wizard prompt if this is a first launch. ✓ Dashboard visible, no errors in the browser console

STEP 3Setup Wizard

Complete the guided setup. Paste credentials from Step 1 when prompted. The wizard validates each connection live before saving.

  • Connect Shopify Paste the dev store URL and access token. Wizard should confirm connection with a green checkmark. ✓ "Connected" confirmation shown
  • Connect Anthropic (Claude) Paste the API key. This powers all AI analysis and copy generation. ✓ API key validated successfully
  • Connect at least one POD provider Enable Gelato, Printify, or Printful (or all three). Paste the credentials from Step 1. ✓ Provider(s) connected, templates visible
  • Configure at least 2 product templates Select product types you want created for each drop (e.g., Fine Art Poster + iPhone Tough Case). For Gelato, you'll need to create templates on gelato.com first, then enter the template IDs here. ✓ Templates saved, shown in config summary
  • Connect Backblaze B2 Paste the B2 credentials. This is where your high-res images are stored. ✓ Bucket connection verified
  • Complete setup wizard Review all settings and save. The dashboard should now show your configured providers and templates. ✓ Setup complete, dashboard shows all connections green

STEP 4Test Drop — Single File

This is the core test. Drop one image and verify the full pipeline works end to end.

Use any PNG, JPG, or TIF file. Art images work best — photographs, illustrations, paintings. Avoid screenshots or memes. The AI analyzes the visual content of the image to write product copy.
  • Start the worker Click "Start Worker" in the dashboard. The live log should show "Worker started. Watching for files..." ✓ Worker running, log visible
  • Drop one image into the incoming folder Drag a PNG, JPG, or TIF file into the ArtDrop incoming folder (shown in the dashboard). Watch the live log — you should see each pipeline step appear in real time. ✓ Log shows: image detected → AI analyzing → copy written → products created → published
  • Verify AI-generated copy Check the log output for the generated title and description. It should describe what's actually in the image, not be generic filler. ✓ Title and description are relevant to the image content
  • Verify products in Shopify Log into the Shopify dev store admin. Go to Products. You should see new products created for each template you configured (e.g., one poster + one phone case). ✓ Products exist in Shopify with correct titles, descriptions, images, and tags
  • Verify SEO metadata Click any product in Shopify admin → scroll to "Search engine listing." SEO title, meta description, and URL handle should all be filled in. ✓ SEO title, meta description, and alt text are all populated
  • Verify product images Each product should have an image attached (the artwork on the product). Check that the image loaded correctly and isn't broken or blank. ✓ Product images display correctly
  • Check the audit log An Excel file (mhp_product_log.xlsx) should be created/updated in the ArtDrop folder. Open it and verify entries for the products just created. ✓ Audit log has entries with thumbnails, metadata, and timestamps

STEP 5Test Drop — Batch

Drop 3 images at once to test batch processing. ArtDrop should process them sequentially.

  • Drop 3 images into the incoming folder at once Use three different images. The log should show them queued and processed one at a time. ✓ All 3 images processed sequentially. Each gets unique copy. No duplicates, no crashes.
  • Verify all products created in Shopify Each image should produce products for all your configured templates. If you have 2 templates and dropped 3 images, you should see 6 new products. ✓ Correct number of products, all with unique titles and descriptions

STEP 6Voice Trainer (Optional)

Test the interactive Voice Trainer. This teaches ArtDrop how you describe your art.

  • Open the Voice Trainer Navigate to the Voice section in the dashboard. Drop an image to see AI-generated copy, then give feedback in plain English. ✓ Voice Trainer UI loads, generates copy, accepts feedback

STEP 7Report Issues

If anything breaks, doesn't work as described, or feels confusing — that's exactly what we need to know. No issue is too small.

How to report: Screenshot the issue + copy the relevant log output. Send both to Michael via text, email, or however he gave you this link. Include which step you were on.

What to include

  • Your OS (Mac or Windows) and version
  • Which step failed
  • Screenshot of the error or unexpected behavior
  • Copy of the live log output (select all text in the log window)

Known Limitations (Not Bugs)

These are expected in the beta and not things to report:

  • Products are created in draft status (not published) — this is intentional
  • Windows build is unsigned — you'll see a "Windows protected your PC" warning. Click "More info" → "Run anyway"
  • Social post generation creates text files but does not auto-post (that's a separate step)
  • Gelato templates must be created on gelato.com before use — ArtDrop references them by ID
  • First drop may take longer (5-8 min) as AI warms up. Subsequent drops are faster (3-5 min)
